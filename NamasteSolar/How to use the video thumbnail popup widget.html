How To Use The Spotlight Widget

Page 1

  • Get video/image url's if uploaded to sitefinity

    1. (#1) - Find the video/image you want in sitefinity (screenshot shows video) and click 'Embed this video'
    2. (#2) - Copy the entire 'Link' text

    *If not hosted on sitefinity, then just use the url to the image or video file. Youtube and other page links will not work, it must be a link to the file itself

  • Widget options

    1. (#1) - In the right pane 'Drag widgets', under 'Custom', drag the 'Video Thumbnail Popup' widget onto the page.
    2. (#2) - Click 'Edit' on the widget.
    3. (#3) - Click the 'categorized' icon as seen in the screenshot
    4. (#3) - Paste the Video Thumbnail and Video URL's into their respective boxes
    5. (#4) - If there are multiple video thumbnails on the page, make sure to input a UniqueIdentifier. This will map the thumbnail to the correct video popup
    6. (#5) - Optional: Define height and widths for the video and thumbnail. If set to zero, they will be set to their defaults (thumbnail: 100x70, video: 500x415)

    * - Note: If you want margins around the placement of the thumbnail, you will have to add a layout element and place the widget inside of that. This documentation assumes knowledge on how to add layout elements.

  • End result

    If either the video url or thumbnail url are missing, then the widget shows up with error placeholder text
