How To Use The Spotlight Widget

Page 1

  • Get video/image url's if uploaded to sitefinity

    1. (#1) - Find the video/image you want in sitefinity (screenshot shows video) and click 'Embed this video'
    2. (#2) - Copy the entire 'Link' text

    *If not hosted on sitefinity, then just use the url to the image or video file. Youtube and other page links will not work, it must be a link to the file itself

  • Widget options

    1. (#1) - While editing the home page, click edit
    2. (#2) - Paste the video and thumbnail url links into their respective text boxes
    3. (#3) - Optional: Put in video heights and widths. If Zero is entered, the default is width:500 height:415.

  • End result

    If either the video url or thumbnail url are missing, then the widget shows up with slightly longer text and the source of the quote below

    If both the video and thumbnail exist however, then the quote source is removed, and the text is slightly shorter.
